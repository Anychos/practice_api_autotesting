# Online Shop Auto-testing API

Это учебный проект в котором я оттачивал практические навыки после прохождения курса по авто-тестированию API

---

## О проекте

В проекте реализована автоматизация тестирования CRUD операций для онлайн магазина с сущностями:
- Authentication
- Users
- Products
- Carts
- Orders

---

## Технологии

В проекте используются:
- Python 3.13  
- Pytest (fixtures, plugins, markers)
- HTTPX (client) 
- Allure (steps, reports, tags, severity, labels) 
- Faker (для генерации тестовых данных)
- Pydantic (для валидации схем запросов и ответов)
- Swagger (для генерации отчетов о покрытии)
- Postgres Data Base (хранение данных)

---

## Особенности реализации

- Проект реализован с применением принципов ООП. В частности инкапсуляция логики и наследование объектов
- Все тесты изолированы и запускаются независимо друг от друга
- Написаны тесты как для позитивных сценариев, так и для негативных
- Все тесты взаимодействуют с базой данных
